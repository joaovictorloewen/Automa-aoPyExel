# 📊 Dashboard de Vendas e Automação de Dados

## 📌 Sobre o Projeto

Este projeto reúne duas etapas fundamentais do processo de análise de dados:

1. **Automação e tratamento de dados utilizando Python**
2. **Construção de Dashboard Interativo no Power BI**

O objetivo foi criar um fluxo completo de Business Intelligence, desde a limpeza da base de dados até a visualização dos indicadores estratégicos em um dashboard executivo.

---

## 🚀 Funcionalidades

### 🐍 Automação de Dados com Python

O script desenvolvido realiza automaticamente:

- Remoção de linhas em branco
- Remoção de registros duplicados
- Padronização dos dados
- Tratamento de inconsistências
- Organização da base para análise
- Geração de uma base pronta para o Power BI

---

### 📊 Dashboard Power BI

#### Visão Geral

- Faturamento Total
- Ticket Médio
- Quantidade Vendida
- Total de Vendedores
- Evolução do Faturamento Mensal
- Distribuição Geográfica das Vendas
- Top Vendedores
- Participação por Categoria

---

#### Análise de Vendedores

- Ranking de vendedores
- Participação nas vendas
- Evolução do desempenho
- Indicadores individuais

---

#### Análise de Produtos

- Produtos mais vendidos
- Faturamento por produto
- Participação por categoria
- Relação entre quantidade e faturamento

---

#### Análise Geográfica

- Vendas por estado
- Ranking regional
- Distribuição geográfica das vendas

---

#### Análise Financeira

- Faturamento acumulado
- Crescimento percentual
- Total de descontos
- Comparação entre receita e descontos
